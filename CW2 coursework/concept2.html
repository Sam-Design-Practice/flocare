CSS Styling

CSS (Cascading Style Sheets) is the language used to style web pages. With CSS, you can control the appearance of your content, including layout, colors, fonts, and more!

Understanding the Basics

CSS allows you to style HTML elements by using selectors, properties, and values.

  • Selectors: Identify the HTML elements you want to style (e.g., `p`, `.class`, `#id`).
  • Properties: Define the styles you want to apply (e.g., `color`, `margin`, `font-size`).
  • Values: Specify the settings for properties (e.g., `red`, `20px`, `center`).

Here's an example:

/* This CSS rule styles all paragraphs to be blue and centered */
p {
    color: blue;
    text-align: center;
}

The CSS Box Model

The box model is a fundamental concept in CSS. It describes how elements are structured:

  • Content: The main content of the box (e.g., text, images).
  • Padding: Space between the content and the border.
  • Border: Surrounds the padding and content.
  • Margin: Space between the element and others around it.

CSS Selectors

CSS selectors target elements to apply styles. Here are some common ones:

  • Universal Selector (`*`): Applies styles to all elements.
  • Type Selector (`h1`, `p`): Targets elements by their HTML tag.
  • Class Selector (`.class`): Targets elements with a specific class.
  • ID Selector (`#id`): Targets elements with a specific ID.
  • Descendant Selector: Targets elements nested within others (e.g., `div p`).

Responsive Design

Responsive design ensures that your website looks good on all devices. Use media queries to apply styles based on the screen size:

/* Example Media Query */
@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
}

In this example, the font size will be reduced for screens smaller than 768px.
