Learn HTML Basics

HTML stands for HyperText Markup Language. It's the standard language used to create web pages.

1. HTML Document Structure

An HTML document has a basic structure:

<!DOCTYPE html>
<html>
    <head>...</head>
    <body>...</body>
</html>

2. Common Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Hyperlink
  • <img>: Image
  • <ul>, <ol>, <li>: Lists
